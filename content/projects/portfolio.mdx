# Building this Portfolio

This portfolio was built using exciting new technologies!

- **Nextra 4** with Next.js App Router for MDX rendering.
- **Tailwind CSS** for rapid styling.
- **Shadcn UI** for accessible and beautiful components.

### Implementation Setup

We integrated **Pages CMS** to seamlessly edit projects and blogs via the GitHub repository directly!
